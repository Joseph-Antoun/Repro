The directory tree is presented below. Each subdirectory contains its own README file named project4_README.txt 
to differentiate from any readme written by the authors of the original paper.

./original_code
    Original GitHub code published by the authors of the paper. Does not work out of the box.
./functional_code
    Modified version of the published original code that works and generates the figures in section 3.b of the write-up.
./project4
    Contains the multiple add-ons and contributions we made either to
        - reproduce some of the results ./reproducing_results
        - perform new experiments       ./ablation_studies
